# Git Analyzer (React + Vite)

Simple GitHub repository analyzer using React + Vite (plain CSS, no Tailwind).

## Quick start

1. Install dependencies
```bash
npm install
```

2. Run dev server
```bash
npm run dev
```

Open the local URL shown by Vite (e.g. http://localhost:5173).

## Notes
- To avoid GitHub rate limits, add a `.env` file with `VITE_GH_TOKEN=your_token` in project root.
